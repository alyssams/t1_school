attributes

We can give tags more instructions by including attributes in the opening tag. An attribute is simply a characteristic or some description for the content in the element. You saw this with src, alt, and title in <img>.


<body>
    <img src="images/cats.jpg" alt="cats like to meow" title="cats like to 
    meow"/>
</body>

indention

indention is the amount each line is spaced in fromthe margin. You'll notice that when we place tags inside of other tags, we indent them more. This helps make your code readable to yourself and others that look at your code.


<!DOCTYPE HTML>
<html>
    <body>
	  <h1>howdy</h1>
	  <p>paragraph goes here!</p>
    </body>
</html>

anchor tags

the <a> allows you to link text or an image to another document. Anchor tags need an attribute called hypertext reference (href) to tell it where to go.

<body>
   <a href="http://www.google.com">Google</a>
   <a href="http://www.google.com"><img src="images/funkitty.jpg"/></a>
</body>

Ordered Lists

<ol> = an ordered list of items.

<li> = a list item.


<body>
   <ol>
     <li>list item 1</li>
     <li>list item 2</li>
     <li>list item 3</li>
   </ol>
</body>
  1. list item 1
  2. list item 2
  3. list item 3

Unordered Lists

<ul> = a collection of items that do not have a numerical ordering, and their order in the list is meaningless.


	    <body>
	    	<ul>
	    	    <li>list item 1</li>
	    	    <li>list item 2</li>
	    	    <li>list item 3</li>
	    	</ul>
	    </body>
  • list item 1
  • list item 2
  • list item 3
