ID & CLASS

In CSS, a class selector is a name preceded by a full stop (“.”) and an ID selector is a name preceded by a hash character (“#”).The difference between an ID and a class is that an ID can be used to identify one element, whereas a class can be used to identify more than one..

<!DOCTYPE HTML>
<html lang="en-US">
    <head>
    	<meta charset="UTF-8">
        <title>Title of page</title>
    </head>
    <body>
    	 <div id="one">Hello</div>
    	 <div class="two">i'm a class!</div>
    	 <div class="two">I'm also a class!</div>
    </body>
</html>

Keep it Classy

Classes are super useful when you want a group of elements to receive the same styling. Instead of using the same rules on different selectors, we just apply it with the same class and all of those elements will get the same styling.


Classes are assigned to HTML elements with the word class and an equals sign, like so:

		
			<div class="square"></div>

Classes are identified in CSS with a dot (.), like so:

		
			.square {
			    height: 100px;
			    width: 100px;
			}

ID is one of a kind

IDs are unique. If you have an element that is one of a kind, and only needed to be used once, using an ID is perfect to use. Only one element will receive a certain kind of styling.

IDs are assigned to HTML elements with the word id and an equals sign:

		
			<div id="one"></div>
			<div id="two"></div>

IDs are identified in CSS with a pound sign (#):

		
			#first {
			    height: 50px;
			}

			#second {
			    height: 100px;
			}

The Box Model

You define internal styles in the head section of an HTML page, by using the <style> tag

margin and padding are the two most commonly used properties for spacing-out elements. A margin is the space outside something, whereas padding is the space inside something.

The margin is the space around the element. The larger the margin, the more space between our element and the elements around it. We can adjust the margin to move our HTML elements closer to or farther from each other.

The border is the edge of the element. It's what we've been making visible every time we set the border property.

The padding is the spacing between the content and the border. We can adjust this value with CSS to move the border closer to or farther from the content.

The content is the actual "stuff" in the box. If we're talking about a element, the "stuff" is the text of the paragraph.

Here is a cool example from learnlayout.com

Border

Borders can be applied to most HTML elements within the body.

border-width sets the width of the border. border-style sets the style. There are other values which include: solid, dotted, grove, ridge, double, etc. border-color sets the color of the border.

		
			.container {
			    border-width: 3px;
			    border-style: dashed;
			    border-color: blue;
			}

			/*another way of writing this:*/

			.container {
				border: 3px dashed blue;
			}

			/*SHORTCUT!*/

Margins

The margin clears an area around an element (outside the border). The margin does not have a background color, and is completely transparent.

"Setting the width of a block-level element will prevent it from stretching out to the edges of its container to the left and right. Then, you can set the left and right margins to auto to horizontally center that element within its container. The element will take up the width you specify, then the remaining space will be split evenly between the two margins." - learn css layout

		
			.container {
				width:50px;
				height:50px;
				margin:0 auto;
			}

			.box {
				margin-top: 10px;
				margin-right: 10px;
				margin-bottom: 10px;
				margin-left: 10px;
			}

			.spotify_container {
				margin: 10px 15px 10px 15px;
			}

			.lego_container {
				margin: 30px 50px;
			}

Padding

The padding clears an area around the content (inside the border) of an element. The padding is affected by the background color of the element.

		
			.container {
				width:50px;
				height:50px;
				padding:20px;
			}

			.box {
				padding-top: 10px;
				padding-right: 10px;
				padding-bottom: 10px;
				padding-left: 10px;
			}

			.spotify_container {
				padding: 10px 15px 10px 15px;
			}

			.lego_container {
				padding: 30px 50px;
			}
