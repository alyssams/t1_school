HTML

  1. What it stands for

    Hyper Text Markup Language

  2. The definition

    A standardized system for tagging text files to achieve font, color, graphic, and hyperlink effects on World Wide Web pages. Hypertext Markup Language forms the structural foundation of any webpage.

  3. We use it everywhere

    Web pages all over the world consist of HTML.

Structure

<!DOCTYPE HTML>
<html>
    <body>
    	<h1>Hello, World!</h1>
    	<p>Scotch Scotchy Scotch!</p>
    </body>
</html>

Images Cont'd

Note that the image tag does not look like this: <img></img>. This is incorrect.

In order to add an image, you will need to reference where the image lives. This is where we get into attributes.

src is short for source. This tells the browser where it can find the image file. This will usually be a relative URL pointing to an image on your own site. Images will live in an images folder you create.

alt is short for alternative. This attribute provides a text description of what the image is.

title is an attribute that provides additional information about the image.

height and width are two attributes that specifies the width and height of the image in pixels. We'll learn more about pixels later.

<img src="images/cat.jpg" alt="kitty sticking tongue out." width="200" height="200"/>
