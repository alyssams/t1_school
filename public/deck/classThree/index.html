div

The <div> element is used a lot in developing. Short for "division," <div> allows you to divide your page into sections (that is, different pieces). It's a block-line element which means you would have a line-break before and after using it.


<!DOCTYPE HTML>
<html>
    <body>
	 <div>
	    <h1>Title</h1>
	    <div>
	        <p>This is a paragraph.</p>
	    </div>
	 </div>
	 <div style="width:50px; height:50px; background-color:blue"></div>
	 <div style="width:50px; height:50px; background-color:green"></div>
	 <div style="width:50px; height:50px; background-color:yellow"></div>
    </body>
</html>

span

The span element is used for a small chunk of HTML inside a line (such as inside a paragraph) therefore it's an inline element. It's similar to the div element such that it holds no meaning.


	    <body>
	    	<div>
	    	   <p><span style="color:blue;">H</span>ello!</p>
	    	</div>
	    </body>

<head>

The head element contains information about our document which won't be visible to visitors.

title element - defines title of the document. Displayed on browser toolbar and in search results.

meta tag - provides information about the page. This information is used by browsers,search engines, social media, etc. It's a self closing tag.

charset attribute - tells browser how the page is formatted and what types of characters are used. The character set is set as utf-8 or universal character set transformation format, eight bit.


	<head>
	    <meta charset="utf-8">
	    <meta name="author" content="Alyssa Saucedo">
	    <meta name="description" content="Class Three of HTML/CSS Course">
	    <meta name="keywords" content="HTML, CSS">
	    <title>
		  HTML/CSS Course Three
	    </title>
	</head>

CSS

what is css?


CSS (which stands for Cascading Style Sheets) is a language used to describe the appearance and formatting of your HTML aka makin' it real pretty.


Cascade defines the order of precedence for how conflicting styles should be applied.


		p {
		  color:blue;
		}
		p {
		  color:green;
		}

CSS Syntax

This syntax is different from the HTML syntax/structure.

A selector can be any HTML elements(p, a, img, div), just make sure to take off the < >.

A property is an aspect of the selector. A value is the setting for that property

To end the property-value, you need to close it with a semi-colon (;).


		selector {
		   property: value;
		}

ALL HTML Elements are Selectors

<style> - internal

You define internal styles in the head section of an HTML page, by using the <style> element


	    <head>
	    	<style>
	    	    h1 {
			font-size:30px;
	    		}
	    	    p{
			color:blue;
	    		}
	    	</style>
	    </head>

style.css - external

Seperates style from markup by linking a stylesheet to the HTML document. This makes our code a lot cleaner to read and more organized.


	    <head>
	    	<link type="text/css" rel="stylesheet" href="stylesheet.css"/>
	    </head>

On most if not all versions of Mac, Windows, etc will have web safe fonts that will be viewable to everyone who goes to your page. Here is a list. CSS has built in defaults

Pixels

A pixel is a dot on your computer screen. Let's say someone asks you to change the font-size of a title. A pixel is universal thus 20px in height will be the same on someone's old macbook. Here is more detailed definition


	    .box {
		width: 30px;
		height: 50px;
	    }
	    p {
		font-size: 20px;
		}

Commenting in CSS
